# 우리 모둠의 혈당 변화 · 2형 당뇨병 예방 실험 웹앱

학생들이 로그인 없이 링크/QR로 접속해, 6개 모둠(3개 비교쌍)이 각각 배정된 음식을 먹고
공복(0분)·식후 30분 혈당을 모둠원 4명이 각자 입력하면, 모둠 평균이 실시간으로 계산되어
다른 모둠·짝 모둠과 비교되는 웹앱입니다.

## 구성

- `index.html` — 학생용 웹앱 (학급 선택 화면 + 대시보드). 정적 파일 하나로 동작하며 GitHub Pages에 그대로 올리면 됩니다.
- `apps-script/Code.gs` — 구글 스프레드시트에 붙이는 Apps Script 백엔드 (무료 DB 역할).

## 실험 구조

- 학급: 아이키커반 / 진격반 / 햇님반 (3개 학급, 각 학급은 독립적으로 데이터가 쌓임)
- 학급마다 6개 모둠이 아래처럼 고정 배정되어 3개의 비교쌍을 이룸

  | 모둠 | 음식 | 비교쌍 |
  |---|---|---|
  | 1모둠 | 콜라 | 콜라 vs 제로콜라 |
  | 2모둠 | 제로콜라 | |
  | 3모둠 | 흰 식빵 | 흰 식빵 vs 통밀빵 |
  | 4모둠 | 통밀빵 | |
  | 5모둠 | 초코파이 단독 | 초코파이 vs 초코파이+견과류 |
  | 6모둠 | 초코파이+견과류 | |

- 모둠마다 학생 4명이 각자 자기 번호 칸에 공복(0분)·식후 30분 혈당을 입력 → 모둠 평균 Δ 자동 계산
- 입력은 버튼을 누르지 않아도 자동 저장되며(약 0.7초 디바운스 후 서버 전송), 같은 반의 다른 기기 화면에도 주기적으로 반영됨

## 1. 구글 스프레드시트 + Apps Script 배포

1. 새 구글 스프레드시트를 만듭니다. (시트 이름은 자유, 스크립트가 `혈당데이터` 탭을 자동 생성합니다)
2. 메뉴에서 `확장 프로그램 → Apps Script`를 엽니다.
3. 기본 `Code.gs` 내용을 지우고, 이 저장소의 `apps-script/Code.gs` 내용을 붙여넣습니다.
4. 우측 상단 `배포 → 새 배포`를 클릭합니다.
   - 유형: **웹 앱**
   - 실행 계정: **나(Me)**
   - 액세스 권한: **전체(Anyone)** ← 학생들이 로그인 없이 접근해야 하므로 반드시 이 설정이어야 합니다.
5. 배포 후 발급되는 `https://script.google.com/macros/s/xxxx/exec` 형태의 URL을 복사합니다.
6. 스프레드시트를 열어보면 `혈당데이터` 탭과 헤더(`반·모둠·학생·공복(0분)·30분후·수정시간`)가 자동으로 생성되어 있는지 확인합니다. (처음 read/write 요청이 한 번 있어야 생성됩니다)

## 2. 웹앱에 Apps Script URL 연결

`index.html` 상단의 `CONFIG.GAS_URL` 값에 5번에서 복사한 URL을 붙여넣습니다.

```js
const CONFIG = {
  GAS_URL: 'https://script.google.com/macros/s/xxxx/exec',
  ...
};
```

이 값을 비워두면 서버 연동 없이 **로컬 계산 모드**로 자동 전환되어, 이 기기(브라우저) 안에서만
데이터가 쌓이고 계산됩니다(테스트용). 실제 수업에서는 반드시 URL을 채워 넣어야 학급 전체
데이터가 공유됩니다.

## 3. GitHub Pages 배포

이 폴더를 GitHub 저장소에 올리고, 저장소 설정의 `Pages`에서 배포 브랜치를 지정하면
`index.html`이 정적 페이지로 서비스됩니다. 배포된 주소를 QR코드로 만들어 학생들에게 공유하세요.

## 동작 방식 요약

- 학생은 링크/QR로 접속 → 학급(3종) 선택 → 대시보드 진입
- 대시보드에서 자기 모둠·자기 번호 칸에 공복(0분) / 30분후 혈당을 입력하면 각 필드 입력 후
  약 0.7초 뒤 자동으로 `GET {GAS_URL}?action=write&data=...` 요청이 전송되어 시트에 반영(upsert)됨
- 대시보드 진입 시 및 8초마다 `GET {GAS_URL}?action=read` 요청으로 전체 데이터를 다시 읽어와
  상단 요약 카드, 6가지 비교 막대그래프, 3개 비교쌍 카드를 갱신
- `결과 복사하기` 버튼은 현재 학급의 입력된 데이터를 탭으로 구분된 텍스트로 클립보드에 복사해
  구글 시트에 바로 붙여넣기 할 수 있게 함
- `이 반의 모든 입력값 지우기` 버튼은 확인 후 해당 학급의 모든 입력을 초기화함(`action=clear`)
- 네트워크 오류나 URL 미설정 시에도 화면이 멈추지 않고, 이 기기에 저장된 데이터만으로 계산을
  이어가는 오프라인 모드로 자연스럽게 전환됨 (읽기/쓰기/초기화 모두 항상 GET 방식만 사용해 CORS 문제 회피)
